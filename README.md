# cs-sky-changer
sky changer .cfg files for cs2 and csgo

Put `skycs2.cfg` in `Counter-Strike Global Offensive/game/csgo/cfg/`
Use the command `exec skycs2` to load it.

Put `skycsgo.cfg` in `Counter-Strike Global Offensive/csgo/cfg/`
Use the command `exec skycsgo` to load it.

Use the command `skyhelp` to get more commands.
